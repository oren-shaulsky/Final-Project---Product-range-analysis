The project contains creating categories by description of the goods using lemmatization, EDA, Analysis of the business metrics, statistical test, creating dashboard on Tableau public and presentation.

Description of the files:

- ecommerce_dataset_us.7z - archive with source data file
- Product range analysis - Jupyter Notebook in .ipynb and .pdf formats with the project description, the project itself and creating "dashboard_data" and "requirements" files
- dashboard_data - file for creating the dashboard 
- Dashboard link - the link for the dashboard on Tableau public
- Presentation - presentation of the project
- requirements - code requirements and libraries needed to downoad to get the code run properly
